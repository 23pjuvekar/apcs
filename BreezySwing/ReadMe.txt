Everything you need to use BreezySwing is here. Installation instructions are given on our website, http://home.wlu.edu/~lambertk/BreezySwing/index.html

BreezySwing.jar              The easiest way to use our packages is to install
                             this jar file.

BreezySwingDoc               This folder contains Java html-style documentation. 
                             The top level page is index.html.

BreezySwingSwingExamples     Demo programs discussed in our website tutorial.

BreezySwingSwingSourceFiles  This folder contains source code for all the classes.


                      
Happy programming :-)
